1.Purpose

We aim to develop a Web application that provides support for interactive learning of concepts and activities specific to gardening. For each specific concept / tool, we will provide the user with information such as meaning, type, practical advice, context of use, plus suggestions on when to perform activities, such as, sowing, irrigation, etc. The system will monitor the progress made by each user in the learning process, providing the user with several structure tests on 3 levels: easy, medium and heavy. Each user must first pass the entire level easily, to have access to the intermediate level, and respectively at the difficult level. Each test will be based on the information present on the web page. Only the one who has mastered all the information well can successfully pass all the levels. It will also generate a ranking of the top 10 most diligent users, also available as an RSS feed, which will be based on the number of points accumulated after taking the tests.

Registered User

  • Log in to the application using a username and password.After logging in, he has the following rights:
    • can access all the information present on the web site divided into categories:Teaching gardening,Ground Soils,Gardening Tools, Landscaping and Plant's Calendar
    • has access to the tests present after studying each subcategory, which are structured on 3 levels: easy, medium and hard
    • can see the ranking of the most diligent users in an RSS feed: TOP 10
    • can see the results of all the tests passed, and the My Progress section
    • can contact the website administrators via email
    • has access to all social links on other platforms

Users without an account

  • By entering the website, he has the following rights:
    • he can sign up by : choosing a pseudonym, entering his first and last name, entering his valid e-mail and choosing a control question with an answer
    • can access all the information present on the web site divided into categories:Teaching gardening,Ground Soils,Gardening Tools, Landscaping and Plant's Calendar
    • can see the ranking of the most diligent users in an RSS feed: TOP 10
    • has access to all social links on other platforms

Admin user

  • By passing all the tests on the web site, this kind of users (admin users) have additional rights:
    • has an additional right: can access the management section where he can delete, insert and update data on the web-site
    • can access all the information present on the web site divided into categories:Teaching gardening,Ground Soils,Gardening Tools, Landscaping and Plant's Calendar
    • has access to the tests present after studying each subcategory, which are structured on 3 levels: easy, medium and hard
    • can see the ranking of the most diligent users in an RSS feed: TOP 10
    • has access to all social links on other platforms
    • can be contacted by all web-site's users via email
